# Clinical Data Analytics Portfolio

Pharmacist transitioning into Clinical Data Analytics, combining healthcare domain expertise with data analysis to generate actionable insights for hospitals, public health systems, and pharmaceutical organisations.

This portfolio demonstrates the application of SQL, healthcare data modelling, and business intelligence tools to analyse clinical outcomes, medication utilisation, and hospital performance.

## Core Focus Areas

• Hospital performance analytics  
• Medication utilisation and pharmaceutical trends  
• Clinical outcome analysis  
• Healthcare KPI reporting  
• Data-driven healthcare decision support  

## Technical Tools

- SQL
- PostgreSQL
- Power BI
- DataCamp DataLab
- Python (in progress)

## Featured Projects

### 1. PBS Medication Utilisation Analysis
Analysis of Pharmaceutical Benefits Scheme dispensing trends across therapeutic drug classes in Australia.

### 2. Hospital Readmission Rate Analysis
Clinical KPI analysis of hospital readmissions using simulated inpatient datasets and SQL cohort logic.

### 3. Pharmacy Inventory Optimisation
Operational analytics project identifying slow-moving medications and expiry risks using SQL window functions.

### 4. Adverse Drug Event Reporting Analysis
Clinical safety analysis of adverse drug event reports with categorisation and trend analysis.

### 5. Healthcare Executive Performance Dashboard
Integrated dashboard analysing hospital admissions, cost, and patient outcomes for executive reporting.
